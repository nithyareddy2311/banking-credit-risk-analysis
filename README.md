📌 Project Overview

This project focuses on analyzing banking customer data to understand financial behavior, identify credit risk patterns, and support data-driven decision-making. It also includes a machine learning model to predict high-risk customers.

🎯 Objectives
Analyze customer demographics and financial activity
Identify key factors influencing credit risk
Segment customers into high-value and high-risk groups
Build a predictive model for credit risk classification

📁 Dataset
Records: 3,000+ customers
Features: 25 columns including:
Estimated Income
Bank Loans
Bank Deposits
Credit Card Balance
Properties Owned
Risk Weighting
🛠 Tools & Technologies
Python (Pandas, NumPy)
Data Visualization: Matplotlib, Seaborn
Jupyter Notebook
Machine Learning: Scikit-learn

🔍 Project Workflow
Data Collection & Storage 
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Data Visualization
Machine Learning Model (Logistic Regression)

🤖 Machine Learning Model
Model Used: Logistic Regression
Goal: Classify customers into High Risk and Low Risk
Target Variable: Risk Category (derived from Risk Weighting)

📊 Model Performance
Accuracy: 89%
AUC Score: 0.87

📈 Key Insights
Higher loan amounts and credit card balances increase credit risk
Customers with lower deposits show higher risk levels
Income alone is not a strong predictor of credit risk
High-value and high-risk customers can be effectively segmented

📊 Visualizations
Correlation Heatmap
Scatter Plots (Loans vs Deposits)
Histograms & Distribution Plots
Boxplots for Risk Analysis


🎯 Business Impact
Helps banks identify high-risk customers early
Supports better credit approval decisions
Improves customer segmentation and targeting
Reduces potential financial losses
